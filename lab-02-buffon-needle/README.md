# Lab 02: Задача Бюффона

## Постановка задачи

На плоскости проведены параллельные прямые на расстоянии `A` друг от друга.
На плоскость случайным образом бросается игла длины `L`. По результатам
`N` бросков требуется получить приближённое значение числа `pi`.

## Теория

Если длина иглы не превышает расстояние между линиями (`L <= A`), вероятность
пересечения иглой одной из линий равна:

$$
P = \frac{2L}{\pi A}
$$

Если в `N` бросках получено `M` пересечений, то:

$$
\frac{M}{N} \approx \frac{2L}{\pi A}
$$

Отсюда приближённое значение числа `pi`:

$$
\hat \pi = \frac{2LN}{AM}
$$

## Параметры эксперимента

У этой работы нет отдельных вариантов. Настройки находятся в `config.py`.

| Параметр | Назначение |
|----------|------------|
| `L` | Длина иглы |
| `A` | Расстояние между линиями |
| `N` | Количество бросков |
| `RANDOM_SEED` | Зерно генератора |
| `MAX_NEEDLES_SHOW` | Количество игл на иллюстрации |

Важно: используемая формула корректна при `L <= A`.

## Вывод

Эксперимент Бюффона показывает связь геометрической вероятности с числом
`pi`. При увеличении числа бросков отношение `M / N` приближается к
теоретической вероятности, а оценка `pi` становится устойчивее.
