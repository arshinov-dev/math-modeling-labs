# Lab 06: Метод хорд

## Постановка задачи

Требуется найти корень уравнения:

$$
f(x) = 0
$$

методом хорд. Формула функции, отрезок и критерии останова берутся из
[конфигурации лабораторной №5](../../lab-05-root-finding-methods/config.py).

Для варианта по умолчанию:

$$
f(x) = \frac{2}{(x^2-x+1)^2} - 1, \qquad x \in [0; 2]
$$

## Теория

На каждой итерации через точки `(a, f(a))` и `(b, f(b))` проводится хорда.
Её пересечение с осью `Ox` определяет новое приближение:

$$
x_n = a - \frac{f(a)(b-a)}{f(b)-f(a)}
$$

После вычисления `x_n` выбирается новый отрезок, на концах которого функция
сохраняет разные знаки.

## Настройка варианта

Общие параметры находятся в `../../lab-05-root-finding-methods/config.py`:

| Параметр | Назначение |
|----------|------------|
| `FUNCTION_FORMULA` | Функция `f(x)` |
| `A`, `B` | Отрезок поиска корня |
| `EPSILON`, `N_MAX` | Критерии останова |

Локальный `config.py` содержит только параметры визуализации метода хорд.

## Вывод

Метод хорд использует значения функции на концах отрезка и не требует
производной. На рассматриваемом варианте последовательность приближений
сходится к тому же корню, что и методы лабораторной №5.
