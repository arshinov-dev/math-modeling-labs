# Lab 03: Задача о разорении игрока

## Постановка задачи

За столом сидят два игрока. У первого находится `X` рублей, у второго `Y`
рублей. Перед ними лежит асимметричная монета: вероятность выпадения орла
равна `P`.

Если выпадает орёл, второй игрок выплачивает первому `1` рубль. Если выпадает
решка, первый игрок платит второму `1` рубль. Максимальное число шагов одной
игры равно `N`, количество моделируемых игр равно `N_GAMES`.

Требуется определить:

1. Вероятность разорения первого игрока.
2. Вероятность разорения второго игрока.
3. Среднюю длину игры.

## Теория

Суммарный капитал игроков неизменен:

$$
T = X + Y
$$

Игра завершается, когда капитал первого игрока достигает одной из границ:

- `0`: первый игрок разорён;
- `T`: второй игрок разорён;
- `N` шагов: игра остановлена по лимиту.

Пусть:

$$
q = 1 - P, \qquad r = \frac{q}{P}
$$

Для неограниченной игры при `P != q` вероятность разорения первого игрока:

$$
P_0 = \frac{r^X - r^T}{1 - r^T}
$$

Для симметричной монеты (`P = q = 0.5`):

$$
P_0 = \frac{T-X}{T}
$$

## Настройка варианта

Параметры находятся в `config.py`.

| Параметр | Назначение |
|----------|------------|
| `P` | Вероятность выигрыша первого игрока |
| `X`, `Y` | Начальные капиталы игроков |
| `N` | Максимальное число шагов одной игры |
| `N_GAMES` | Количество моделируемых игр |
| `RANDOM_SEED` | Зерно генератора |

При малом `N` часть игр может завершиться по лимиту. Теоретические значения
в отчёте относятся к игре без ограничения по числу шагов.

## Вывод

Моделирование позволяет оценить вероятности разорения и среднюю длину игры.

теоретическим значениям.
