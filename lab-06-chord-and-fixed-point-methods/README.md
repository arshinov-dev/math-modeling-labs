# Lab 06: Методы хорд и простых итераций

## Постановка задачи

Требуется найти приближённые решения уравнений двумя методами:

1. Метод хорд для уравнения `f(x) = 0`.
2. Метод простых итераций для уравнения `x = phi(x)`.

Чтобы расчёты и графики не смешивались, методы находятся в отдельных
каталогах:

| Раздел | Документация |
|--------|--------------|
| Метод хорд | [chord-method/](chord-method/) |
| Метод простых итераций | [fixed-point-iteration/](fixed-point-iteration/) |

## Связь с лабораторной №5

Метод хорд использует формулу `f(x)`, отрезок `[A; B]` и критерии останова
из [конфигурации лабораторной №5](../lab-05-root-finding-methods/config.py).

Метод простых итераций использует собственную формулу `phi(x)`, но критерии
останова `EPSILON` и `N_MAX` также получает из лабораторной №5.

## Вывод

Метод хорд не требует вычисления производной и уточняет корень по секущим.
Метод простых итераций удобен после преобразования уравнения к виду
`x = phi(x)`, если на выбранном интервале выполняется условие сжатия.
