# Lab 04: Численные методы интегрирования

## Постановка задачи

Требуется вычислить определённый интеграл:

$$
S = \int_A^B f(x)\,dx
$$

и сравнить следующие численные методы:

| Обозначение | Метод |
|-------------|-------|
| ЛПР | Левые прямоугольники |
| ППР | Правые прямоугольники |
| СПР | Средние прямоугольники |
| ТР | Трапеции |
| ММК | Монте-Карло |
| СИМП | Симпсон |

Для варианта по умолчанию:

$$
f(x) = (x + 1)\sqrt{1-x}, \qquad x \in [0; 1]
$$

$$
S = \frac{14}{15} \approx 0.9333333333
$$

## Теория

Отрезок `[A; B]` разбивается на части. Различие методов заключается в способе
приближения площади каждой части:

- прямоугольники используют значение функции в левой, правой или средней точке;
- трапеции соединяют соседние узлы отрезками;
- метод Симпсона использует параболическое приближение;
- Монте-Карло вычисляет среднее значение функции в случайных точках.

Для оценки качества вычисляются:

$$
\Delta = |\hat S - S_{\text{эт}}|
$$

$$
\delta = \frac{\Delta}{|S_{\text{эт}}|} \cdot 100\%
$$

Каждый метод дополнительно уточняется увеличением числа разбиений, пока не
выполнится условие `Δ < EPSILON` или не будет достигнут `N_MAX`.

## Настройка варианта

Параметры находятся в `config.py`.

| Параметр | Назначение |
|----------|------------|
| `FUNCTION_FORMULA` | Подынтегральная функция |
| `A`, `B` | Границы интегрирования |
| `REFERENCE_AREA` | Эталон (`None` для автоматического уточнения Симпсоном) |
| `EPSILON`, `N_MAX`, `INITIAL_N` | Критерии останова |
| `N_VALUES` | Значения `n` для сравнительной таблицы |

Поддерживаются: `x`, `+`, `-`, `*`, `/`, `^`, скобки, `sin`, `cos`, `tan`,
`exp`, `log`, `sqrt`, `abs`, `pi`, `e`.

## Вывод

Для гладких функций методы средних прямоугольников и Симпсона обычно сходятся
быстрее простых прямоугольников. Метод Монте-Карло универсален, но его результат
имеет статистический разброс и требует большего числа точек.
